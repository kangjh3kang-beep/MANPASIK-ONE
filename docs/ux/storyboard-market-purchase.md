### 인터랙션
- 상단 배너: 자동 캐러셀 (3초 간격), 좌우 스와이프
- 카테고리 아이콘: 탭 시 해당 카테고리 상품 목록으로 이동
- 상품 카드: 탭 시 상품 상세 페이지로 이동
- 장바구니 뱃지: 담긴 상품 수 실시간 표시
- 검색: 카트리지 이름, 카테고리, 용도 키워드 검색

### 음성 안내 (TTS)
> "마켓에 오신 것을 환영합니다. 카테고리를 선택하거나 원하는 카트리지를 검색하세요."

### 기능 설명
- ShopService.ListProducts RPC로 인기 상품 조회
- 구독 티어별 가격 차등 표시 (Pro 회원 할인가)
- 카트리지 호환성: 사용자 보유 리더기와 호환 여부 자동 표시

### 인터랙션
- 이미지 갤러리: 좌우 스와이프, 핀치 줌
- 호환성 자동 확인: 사용자 등록 리더기와 NFC 코드 매칭
- "Pro 회원가": 현재 구독 티어에 따라 할인가 표시/미표시
- 리뷰 섹션: 더보기 탭 시 전체 리뷰 목록
- 장바구니 담기: 수량 선택 바텀시트 + 햅틱 피드백
- 바로 구매: 결제 화면으로 즉시 이동

### 에러 케이스
- 리더기 미등록 시: "리더기를 먼저 등록해주세요" 안내 + 기기 관리 이동 버튼
- 재고 없음: "품절" 표시 + 입고 알림 신청 버튼
- 비호환 리더기: "이 카트리지는 v2.0 이상 리더기가 필요합니다" 경고

### 인터랙션
- 토스페이 선택 시: Toss SDK 결제 웹뷰 호출
- 결제 진행: 로딩 오버레이 + "결제 처리 중..." 표시
- 결제 성공: PaymentService.ConfirmPayment RPC 호출 → 주문 완료 화면
- 결제 실패: 에러 메시지 + 재시도 버튼

### 에러 케이스
- 결제 시간 초과 (30초): "결제 시간이 초과되었습니다. 다시 시도해주세요."
- 잔액 부족: "잔액이 부족합니다. 다른 결제 수단을 선택해주세요."
- 네트워크 오류: "네트워크 연결을 확인해주세요." + 오프라인 장바구니 보존

## 접근성 고려사항
- 모든 상품 이미지에 alt 텍스트 제공
- 가격 정보 스크린리더 읽기: "49,000원, 프로 회원 할인가 39,200원"
- 결제 버튼 최소 터치 영역 48×48dp
- 색상 대비 WCAG AA 준수 (가격 텍스트 4.5:1 이상)
