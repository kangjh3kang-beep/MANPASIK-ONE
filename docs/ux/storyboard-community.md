## 장면 1: 커뮤니티 메인 (건강 토픽별 게시판)

### 와이어프레임
```
┌─────────────────────────────┐
│  ◀  커뮤니티       🔍  🔔   │
├─────────────────────────────┤
│ [전체] [혈당] [다이어트] [운동]│
│ [레시피] [정신건강] [Q&A]    │
├─────────────────────────────┤
│ 🔥 인기 게시글               │
│ ┌─────────────────────────┐ │
│ │ 혈당 관리 3개월 후기 🏷혈당│ │
│ │ 건강마스터 · 2시간 전     │ │
│ │ "만파식으로 매일 측정하면서│ │
│ │  공복혈당이 130→98로..."  │ │
│ │ ❤️ 234  💬 45  👁 1.2K   │ │
│ ├─────────────────────────┤ │
│ │ 당뇨 식단 레시피 모음      │ │
│ │ 🏷레시피                  │
│ │ 요리왕 · 5시간 전         │ │
│ │ "혈당 스파이크 없는 점심..."│ │
│ │ ❤️ 189  💬 32  👁 980    │ │
│ ├─────────────────────────┤ │
│ │ 운동 후 혈당 변화 실험     │ │
│ │ 🏷운동 🏷혈당             │ │
│ │ 런닝맨 · 어제             │ │
│ │ "30분 걷기 vs 달리기..."  │ │
│ │ ❤️ 156  💬 28  👁 750    │ │
│ └─────────────────────────┘ │
│                             │
│           [더 보기]          │
├─────────────────────────────┤
│         ✏️ (글쓰기 FAB)      │
├─────────────────────────────┤
│  🏠   📊   ⬡   🛒   ⚙️   │
└─────────────────────────────┘
```

### 인터랙션
- 토픽 탭: 수평 스크롤, 선택 시 해당 카테고리 필터링
- 게시글 카드: 탭 시 상세 보기, 길게 누르면 북마크/신고
- 좋아요: 더블탭 또는 ❤️ 버튼
- FAB (글쓰기): 새 게시글 작성 화면으로 이동
- CommunityService.ListPosts RPC로 게시글 조회 (페이지네이션)

### 음성 안내 (TTS)
> "커뮤니티입니다. 건강 토픽별 게시글을 확인하세요. 글쓰기 버튼으로 경험을 공유할 수 있습니다."

### 인터랙션
- 인증하기: 오늘의 측정 기록 자동 확인 → 인증 완료 애니메이션
- 참여하기: 챌린지 규칙 확인 → 참여 동의
- 순위표: 탭 시 전체 참가자 랭킹 보기
- 보상 포인트: 마켓에서 사용 가능한 적립금
- 게이미피케이션: 레벨 시스템 (뉴비→초보→중급→고급→마스터)

### 인터랙션

- 전문가 답변: ✅ 인증 배지, 상단 고정
- 답변 알림: 내 질문에 답변 시 푸시 알림
- 익명 질문: 닉네임 표시 (실명 비공개)

### 인터랙션
- 익명 공유: 데이터 비식별화 처리 (이름, 나이, 성별 제거)
- 실명 공유: 닉네임 + 연령대 표시
- 데이터 선택: 체크박스로 공유 범위 제어
- 미리보기: 다른 사용자에게 보이는 화면 미리 확인
- CommunityService.CreatePost RPC + HealthRecordService 데이터 첨부
- 게시 후: 좋아요/댓글 알림 설정

### 에러 케이스
- 부적절한 내용: AI 기반 콘텐츠 필터링 → "커뮤니티 가이드라인에 위반되는 내용입니다."
- 의료 조언 경고: 측정 데이터 공유 시 "이 정보는 의료 조언이 아닙니다" 면책 고지

## 접근성 고려사항
- 게시글 이미지: alt 텍스트 필수
- 좋아요/댓글 수: 스크린리더 "좋아요 234개, 댓글 45개"
- FAB 글쓰기 버튼: 접근성 라벨 "새 게시글 작성"
- 챌린지 진행률: "60% 완료, 30일 중 18일 달성"
