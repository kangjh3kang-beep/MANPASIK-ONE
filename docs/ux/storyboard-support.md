## 장면 1: 고객 지원 센터 (Help Center)

### 와이어프레임
```
┌─────────────────────────────┐
│  ◀  고객 지원               │
├─────────────────────────────┤
│ 무엇을 도와드릴까요?          │
│ [    검색어를 입력하세요    ] │
│                             │
│ 자주 묻는 질문 (FAQ)         │
│ ┌─────────────────────────┐ │
│ │ Q. 리더기 연결이 안 돼요   │ │
│ │ A. 블루투스 설정을 확인...  │ │
│ ├─────────────────────────┤ │
│ │ Q. 카트리지 유효기간은?    │ │
│ │ A. 제조일로부터 1개월입...  │ │
│ └─────────────────────────┘ │
│                             │
│ 문의하기                     │
│ ┌───┐ ┌───┐ ┌───┐ ┌───┐  │
│ │🗨️ │ │✉️ │ │📢 │ │📄 │  │
│ │봇  │ │1:1 │ │공지│ │약관│  │
│ └───┘ └───┘ └───┘ └───┘  │
│                             │
│ 📞 고객센터 1588-XXXX        │
│    (평일 09:00 ~ 18:00)     │
└─────────────────────────────┘
```

### 인터랙션
- **검색**: 키워드 입력 시 FAQ 실시간 필터링.
- **FAQ 확장**: 질문 탭 시 답변 내용 아코디언 펼침.
- **1:1 문의**: 탭 시 문의 작성 화면으로 이동.
- **전화 연결**: 탭 시 다이얼러 호출.

## 장면 2: 1:1 문의 작성

### 와이어프레임
```
┌─────────────────────────────┐
│  ◀  문의하기         [제출]  │
├─────────────────────────────┤
│ 문의 유형 선택                │
│ [ 기기 오류       ▼ ]       │
│                             │
│ 제목                         │
│ [ 리더기 전원이 안 켜져요   ] │
│                             │
│ 내용                         │
│ [ 충전을 1시간 해도 반응이    ] │
│ [ 없습니다. LED도 안...     ] │
│                             │
│ 사진 첨부 (선택)             │
│ [ + ] [ img1 ]             │
│                             │
│ 답변 알림                    │
│ [v] 푸시 알림 받기           │
│ [ ] 이메일로 받기            │
└─────────────────────────────┘
```

### 인터랙션
- **유형 선택**: 바텀시트에서 선택 (기기/카트리지/앱/결제/기타).
- **사진 첨부**: 갤러리/카메라 호출.
- **제출**: 유효성 검사 후 서버 전송 → 접수 완료 토스트 메시지.

## 장면 3: 긴급 대응 및 안전 설정 (Safety)

### 와이어프레임
```
┌─────────────────────────────┐
│  ◀  안전 및 긴급 설정        │
├─────────────────────────────┤
│ 긴급 연락망 관리              │
│ [ + 연락처 추가 ]            │
│ • 아내 (010-1234-5678)      │
│ • 아들 (010-9876-5432)      │
│                             │
│ 위험 감지 시 자동 행동        │
│                             │
│ [v] 긴급 연락처에 위치 전송   │
│     "심각한 이상 징후 발생 시" │
│                             │
│ [ ] 119 자동 음성 신고       │
│     "응답 없음 3분 지속 시"   │
│                             │
│ 안전 모드 (독거/고령자)       │
│ [ 설정하기 ]                 │
│ • 일정 시간 미활동 시 알림    │
│ • 야간 낙상 감지 (웨어러블)   │
└─────────────────────────────┘
```

### 인터랙션
- **연락처 추가**: 주소록 연동 또는 직접 입력.
- **자동 행동 토글**: 활성화 시 권한(SMS/위치/통화) 확인 및 동의 절차.
- **119 신고**: 오작동 방지를 위한 2중 확인 팝업 (정말 설정하시겠습니까?).
