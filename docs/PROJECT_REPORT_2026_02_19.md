# 만파식적(Manpasik) 프로젝트 개발 보고서 (2026-02-19)

## 1. 개요 (Overview)
본 보고서는 만파식적 통합 모니터링 대시보드의 최신 개발 진행 상황, 주요 개선 사항, 기술적 구현 내용 및 향후 계획을 포괄적으로 다룹니다. 특히 최근 집중적으로 진행된 **UI/UX 고도화**와 **시스템 안정성 확보**에 초점을 맞추었습니다.

## 2. 주요 개선 사항 (Key Enhancements)

### 2.1. 대시보드 레이아웃 혁신 (Hub & Spoke)
*   **기존 문제**: 리더기(Device)들이 무작위로 배치되어 산만하고, 중앙 지구본과의 연결성이 부족해보였습니다.
*   **개선 결과**:
    *   **중앙 집중형 배치**: 지구본을 중심으로 모든 리더기가 완벽한 원형 궤도(Concentric Orbit)를 그리며 정렬되도록 변경했습니다. 기기가 많을 경우 이중 궤도로 자동 분산됩니다.
    *   **표면 연결성 (Surface Integration)**: 연결선이 지구본의 중심이 아닌 **표면**에서 뻗어 나오도록 하여, 실제 기기가 시스템에 '플러그인' 된 듯한 입체감을 구현했습니다.

### 2.2. 인체 홀로그램 고도화 (Eco-Bio Hologram)
*   **기존 문제**: 인체 시각화가 단순하고 조악하여 몰입감을 저해했습니다.
*   **개선 결과**:
    *   **HoloBody 위젯 개발**: 수천 개의 3D 파티클(입자)로 구성된 고품질 인체 홀로그램을 구현했습니다.
    *   **동적 생체 효과**: 심장과 뇌 부위의 맥동(Glow) 효과와 전신 스캔 애니메이션을 추가하여 살아있는 생체 데이터를 시각화했습니다.
    *   **바이오 탭 신설**: '전체/기체/환경' 외에 **'바이오'** 탭을 신설하여, 선택 시 중앙 홀로그램이 지구본에서 인체 모형으로 자연스럽게 전환됩니다.

### 2.3. 시스템 안정성 및 반응형 강화
*   **UI 크래시 해결**: 탭 전환 시 발생하던 `RenderObject` 관련 오류(UI 깨짐/멈춤)를 렌더링 트리의 고유 키(`Key`) 할당을 통해 완벽하게 해결했습니다.
*   **반응형 디자인**: 윈도우 크기에 따라 배치와 구성 요소 크기가 유동적으로 조절되어, 작은 화면에서도 정보가 잘리거나 겹치지 않도록 개선했습니다.

## 3. 기술적 세부 사항 (Technical Details)

### 3.1. 모듈 구조
*   `MonitoringDashboardScreen`: 상태 관리 및 전체 레이아웃을 담당하는 메인 스크린.
*   `HoloGlobe` & `HoloBody`: `CustomPainter`와 파티클 시스템을 활용한 고성능 시각화 위젯.
*   `OrbitalDeviceNode`: 개별 기기 정보를 표시하고 인터랙션을 처리하는 노드 위젯.

### 3.2. 성능 최적화
*   **CustomPainter**: 복잡한 애니메이션을 `CustomPainter`로 직접 구현하여 렌더링 오버헤드를 최소화했습니다.
*   **Determinisitic Jitter**: 기기의 고유 ID를 기반으로 위치 난수를 생성하여, 화면 갱신 시에도 기기 위치가 흔들리지 않고 고정되도록 했습니다.

## 4. 사용자 가이드 (User Guide)

1.  **대시보드 진입**: 메인 화면에서 '전체 리더기 현황 보기'를 클릭합니다.
2.  **탭 탐색**: 상단의 탭(전체/기체/환경/바이오)을 눌러 필터링된 기기 목록을 확인합니다.
    *   **바이오 탭**: 중앙의 지구본이 인체 홀로그램으로 변경되며, 바이오 관련 센서 데이터가 시각화됩니다.
3.  **기기 상세 보기**: 궤도상의 기기 아이콘을 클릭하거나, 하단 패널을 드래그하여 기기 목록에서 선택하면 상세 정보가 표시됩니다.
4.  **경고 확인**: 기기에 문제가 발생하면 붉은색 경고 배너가 표시되며, 클릭 시 해당 기기로 즉시 이동합니다.

## 5. 향후 계획 (Future Roadmap)
*   **3D 인터랙션 강화**: 홀로그램을 드래그하여 회전하거나 확대/축소하는 기능 추가.
*   **실시간 데이터 연동**: 백엔드 소켓 연결을 통해 센서 수치 변화를 실시간으로 파티클 색상/속도에 반영.
*   **AR 모드 도입**: 모바일 기기에서 카메라를 통해 실제 공간에 홀로그램을 띄우는 기능 검토.
