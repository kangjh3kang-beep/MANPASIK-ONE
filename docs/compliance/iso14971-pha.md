# ISO 14971 사전 위해 분석 (Preliminary Hazard Analysis)

## ManPaSik (만파식) 차동측정 기반 헬스케어 AI 생태계

**문서 번호**: MPS-PHA-001
**버전**: 1.0
**작성일**: 2026-02-14
**참조**: MPS-RMP-001, MPS-FMEA-001, ISO 14971:2019 Annex C

---

## 1. 목적

시스템 설계 초기 단계에서 주요 위해를 식별하고 초기 위험 수준을 평가하여,
상세 설계 단계의 FMEA/FTA에 우선순위 입력을 제공합니다.

---

## 2. 시스템 설명

ManPaSik은 차동측정 기술 기반의 가정용 IVD(체외진단) 플랫폼으로:
- **리더기**: 88~1792채널 센서 어레이 + BLE 5.0 + NFC 리더
- **카트리지**: 29~65,536종 확장 가능 (2-byte 레지스트리)
- **모바일 앱**: Flutter 기반 (iOS/Android)
- **클라우드**: 22+ MSA 서비스, AI/ML 파이프라인
- **의도된 사용**: 자가 건강 모니터링, 의료 전문가 원격 상담 보조
